=======================================
Analytic Tag Dimension Purchase Warning
=======================================

This module extends the functionality of analytic_tag_dimension to have the
warning appear while saving purchase orders, if two tags from the same
dimension are used.

Installation
============

This module will auto install if both analytic_tag_dimension and purchase
modules are installed.

Configuration
=============

Nothing to do if you already configured analytic_tag_dimension
